# Local Configuration for Vim

All you need to do is clone this repository and then create a symbolic
link to the configuration.

```
mkdir -p ~/.vim; cd ~/.vim
git clone https://github.com/zer0-f0x/vim-config.git
chmod +x install
./install
```
